## 🗺️ Interview Roadmap

### Junior/Entry Level (0-2 years)

**Trọng tâm:** SQL, Basic modeling, ETL concepts, Python basics   
**Prep:** 01, 02 (SQL focus), 05 (Python tasks)

### Mid Level (2-5 years)

**Trọng tâm:** Advanced SQL, System design basics, Pipeline optimization   
**Prep:** 02, 03, 04, 05 (all tasks)

### Senior Level (5+ years)

**Trọng tâm:** Complex system design, Architecture decisions, Leadership   
**Prep:** 03 (deep), 04 (STAR stories), 01 (concepts)
