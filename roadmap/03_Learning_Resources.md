## 📖 Sách Quan Trọng

### Must-Read (Bắt buộc đọc)

**1. Designing Data-Intensive Applications (DDIA)**
- Author: Martin Kleppmann
- Focus: Distributed systems, data storage, processing
- Level: Mid → Senior
- Note: "Bible" của Data Engineering

**2. The Data Warehouse Toolkit (3rd Ed)**
- Author: Ralph Kimball
- Focus: Dimensional modeling
- Level: Junior → Mid
- Note: Foundation cho data modeling

**3. Fundamentals of Data Engineering**
- Author: Joe Reis & Matt Housley
- Focus: Modern DE landscape
- Level: Junior → Senior
- Note: Tổng quan toàn diện nhất 2025

### Advanced Reading

**4. Streaming Systems**
- Author: Tyler Akidau, Slava Chernyak
- Focus: Stream processing theory
- Level: Mid → Senior

**5. Data Mesh**
- Author: Zhamak Dehghani
- Focus: Decentralized data architecture
- Level: Senior → Staff

**6. Building Machine Learning Pipelines**
- Author: Hannes Hapke, Catherine Nelson
- Focus: MLOps, feature engineering
- Level: Mid → Senior

### Programming

**7. Fluent Python**
- Author: Luciano Ramalho
- Focus: Advanced Python
- Level: Mid → Senior

**8. Learning Spark (2nd Ed)**
- Author: Jules S. Damji et al.
- Focus: Apache Spark
- Level: Junior → Mid


## 🎯 Learning Paths by Goal

### Path 1: Get First DE Job

**Duration: 6 months**

```
Month 1-2: Fundamentals
- SQL (Mode tutorial + LeetCode)
- Python basics (DataCamp or Real Python)
- Git (version control)

Month 3-4: Core Tools
- One database (PostgreSQL)
- Basic ETL with Python
- Cloud basics (AWS/GCP free tier)

Month 5: First Projects
- Build 2-3 portfolio projects
- Document on GitHub
- Write blog posts

Month 6: Job Search
- Polish resume
- Practice interviews
- Apply broadly
```

### Path 2: Junior to Mid Transition

**Duration: 12 months**

```
Quarter 1: Deepen Core Skills
- Advanced SQL (window functions)
- Spark fundamentals
- Airflow or Dagster

Quarter 2: Modern Data Stack
- dbt mastery
- Data quality tools
- Cloud data warehouse

Quarter 3: Streaming Intro
- Kafka basics
- Simple streaming pipeline
- Exactly-once concepts

Quarter 4: Soft Skills
- Take ownership of projects
- Mentor new team members
- Present to team
```

### Path 3: Senior Preparation

**Duration: 24 months**

```
Year 1: Technical Breadth
- Read DDIA (6 months to absorb)
- System design practice
- Multiple technology stacks
- Architecture documentation

Year 2: Leadership + Depth
- Lead projects
- Mentor juniors
- Expert in 1-2 areas
- External presence (blog, talks)
```
