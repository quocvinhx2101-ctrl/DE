## 💡 PHẦN 2: ĐÁP ÁN & GIẢI THÍCH (KIỂM TRA CẠM BẪY)

1. **B.** Medallion architecture. (Bronze = Raw, Silver = Cleaned, Gold = Aggregated/BI).
2. **B.** `RESTORE TABLE ... TO TIMESTAMP AS OF`. (Bẫy: Revert/Rollback không phải cú pháp SQL chuẩn của Delta).
3. **C.** `VACUUM table_name RETAIN 168 HOURS`. (168 giờ = 7 ngày. Lệnh này xóa các file không còn xài trong log).
4. **C.** `UNION`. (Bẫy kinh điển: Nếu bảo *không duplicate* thì dùng UNION. Nếu dùng UNION ALL nó sẽ giữ tất cả bản ghi lặp).
5. **B.** `explode()`. (Dùng để tung array ra thành từng row riêng biệt. Array functions là keyword quen thuộc cho JSON).
6. **B.** 5. (Bẫy Đếm: `COUNT(col)` sẽ bỏ qua giá trị Null. `COUNT(*)` mới đếm thành 6).
7. **B.** `MERGE INTO`. (Sử dụng cho logic "Có thì update, Chưa có thì insert" - Upsert).
8. **B.** Temporary = session, Global = cluster-wide.
9. **B.** `if day_of_week == 1 and is_weekend:` (Bẫy Python Syntax: Ép kiểu boolean chỉ cần ghi `and variable`, đừng ghi `== "True"`).
10. **C.** `spark.sql()` (Entrypoint để chạy chuỗi SQL trực tiếp trong PySpark).
11. **C.** `spark.readStream()` (Hàm chuẩn của Structured Streaming).
12. **C.** `Trigger.AvailableNow` (Sẽ đọc hết data mới tính đến hiện tại, xử lý xong rồi chủ động tắt máy. Tiết kiệm nhất cho batch).
13. **B.** `format("cloudFiles")` (Đây là cú pháp độc quyền để gọi Auto Loader. Đừng bị lừa chọn `format("autoLoader")`).
14. **B.** `cloudFiles.schemaLocation` (Dùng để bắt buộc lưu trữ schema history, phục vụ schema evolution).
15. **C.** Source Code (At least one Notebook or Library). (Bẫy: Target DB hay Storage là tuỳ chọn, Databricks tự cấp default được).
16. **C.** `SELECT * FROM STREAM(LIVE.raw_events)`. (Biết ngay là DLT vì cái tag `STREAM()` bọc lấy `LIVE.table`).
17. **B.** `EXPECT (...) ON VIOLATION DROP ROW`. (Drop Row = vứt bỏ dòng dởm, tiếp tục cho pipeline chạy. Fail Update = Đứng pipeline).
18. **C.** Pipeline UI -> data quality statistics. (Nhấn thẳng vào cái Table cần xem, UI sẽ hiện thông số Quality).
19. **B.** Cron syntax. (Cách duy nhất để schedule programmatic 100%).
20. **B.** "Repair the task" (Bẫy tối ưu hóa: Đừng chạy lại "từ đầu" gây tốn tiền).
21. **A.** Add Task A to "Depends on" of Task B. (Bịch ngược lại: B phải chờ A xong mới được chạy -> B depends on A).
22. **B.** Job Cluster (Compute sinh ra rồi chết đi lẹ, rẻ nhất cho Daily ETL Batch).
23. **B.** Databricks Asset Bundles (DABs) + Git. (Standard CI/CD mới nhất của Databricks 2026).
24. **C.** `databricks.yml` (Trái tim của hệ thống config).
25. **C.** Resolve merge conflicts. (Phải sửa resolve ở GitHub/GitLab UI, không sửa trực tiếp ở local Databricks được).
26. **B.** Narrow the filters / upsize worker nodes. (Bẫy cực nguy hiểm: Tuyệt đối KHÔNG LỌT VÀO VÒNG Cache(), vì data to đem đi cache bằng RAM càng gây sập OOM nặng hơn).
27. **B.** From the query's page... (Bẫy: Schedule refresh báo cáo nằm ở màn hình lưu Query, không phải ở config SQL Warehouse).
28. **B.** Increase Scaling Range. (Nhiều queries song song: cần tăng Scaling min/max để phân luồng hàng đợi).
29. **A.** Create Alert with Webhook destination. (Cách native nhất của Databricks SQL để bắn notification qua Slack/Teams).
30. **B.** Giao diện Data Explorer, check Owner field.
31. **C.** `SELECT`. (Quyền bé nhất, an toàn nhất để read-only).
32. **B.** Unity Catalog. (Được thiết kế sinh ra để quản trị bảo mật 1 điểm tập trung (Fine-grained AC)).
33. **B.** Data bay màu luôn. (Định nghĩa bảng nằm mặc định trong catalog -> Managed Table. Xóa bảng là xóa CẢ file dưới storage).
34. **C.** Sharing identifier của Unity Catalog metastore.
35. **C.** Trạm trung chuyển Cloudflare R2. (Keyword giảm Networking Cost Cross-Cloud kinh điển nhất).
36. **C.** All dependencies: notebooks, tables, AND reports/dashboards. (Lineage của UC phủ sóng toàn vẹn).
37. **C.** Lakehouse Federation. (Đọc dữ liệu từ DB truyền thống mà không copy).
38. **B.** Không. (Giao thức Delta Sharing gốc là cấp quyền READ-ONLY với bên nhận external).
39. **A.** `to_date()`. (Bẫy: cast_date không có thật).
40. **B.** JSON, CSV, PARQUET, AVRO, và ORC. (AutoLoader rất xịn hỗ trợ nhiều định dạng cho batch raw).
